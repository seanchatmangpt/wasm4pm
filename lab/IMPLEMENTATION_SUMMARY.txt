  Latency:       <1000ms (all profiles)
  Consistency:   ±10% variance
  Assertion:     assert(latencyMs < 1000)

Medium Logs (10K-100K events):
  10K:           <5000ms
  100K:          <30000ms (quality may exceed)
  Memory:        reasonable bounds
  Assertion:     assert(latencyMs < 5000 || 30000)

Scaling Analysis:
  Growth Ratio:  <1.3 (linear O(n))
  Complexity:    Not O(n²)
  Assertion:     assert(growthRatio < 1.3)

Profiles:
  Ordering:      fast ≤ balanced ≤ quality
  Assertion:     assert(fast <= balanced && balanced <= quality)

Memory:
  Growth:        Linear with input size
  Pattern:       Not quadratic or worse
  Release:       Complete after algorithm

Repeatability:
  Variance:      ±10% across runs
  Degradation:   None over time
  Isolation:     Runs don't interfere
