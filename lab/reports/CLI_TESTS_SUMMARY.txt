=================================================================================
                         SUPPORT & TROUBLESHOOTING
=================================================================================

ISSUE: "pmctl: command not found"
  FIX: Install CLI globally or build locally
       npm install -g @wasm4pm/pmctl
       OR npm run build:cli

ISSUE: Permission denied errors
  FIX: Check /tmp permissions
       chmod 755 /tmp
       rm -rf /tmp/pmctl-*

ISSUE: Tests timeout
  FIX: Increase timeout in vitest.config.ts
       testTimeout: 15000

ISSUE: JSON parse errors
  FIX: Run with verbose flag to see output
       npm run test:verbose

ISSUE: Configuration fixture not found
  FIX: Fixtures are in /lab/fixtures/cli-test-configs/
       Tests create temp copies automatically
