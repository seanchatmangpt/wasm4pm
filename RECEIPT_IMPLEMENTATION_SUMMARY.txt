================================================================================
USAGE EXAMPLES
================================================================================

Basic Receipt Creation:
  const receipt = new ReceiptBuilder()
    .setConfig(config)
    .setInput(input)
    .setPlan(plan)
    .setTiming(start, end)
    .setStatus('success')
    .setSummary({traces: 100, objects: 500, variants: 10})
    .setAlgorithm({name: 'alpha++', version: '1.0'})
    .setModel({nodes: 42, edges: 156})
    .build();

Validation:
  const result = validateReceipt(receipt);
  if (!result.valid) console.error(result.errors);

Tampering Detection:
  const isTampered = detectTampering(receipt, config, input, plan);

Hashing:
  const hash = hashConfig(config);
  const isValid = verifyHash(config, hash);
